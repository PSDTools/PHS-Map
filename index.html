How to use!

  1. Press the button below to add a schedule
  2. Type in the number of classes in that schedule
  3. Press "Submit"
  4. Fill out the class information
  5. Press "Save Courses"
  6. Now use the passing time button in-between classes to show a path between those classes!
  7. Use the "Level 1" and "Level 2" buttons below the map to view different floors Click Here for a Video Tutorial!
  8. Now use the "Show Path" button in-between classes to show a path between those classes!
  9. Use the "Level 0", "Level 1", and "Level 2" buttons below the map to view different floors
